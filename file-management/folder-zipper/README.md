# Folder Zipper v1.2

## What does it do?
* Creates ZIP archives of all folders.

## How do I run it?
* Click [here](/file-management/folder-zipper/exe/folder-zipper.exe) and download the script into the folder containing subfolders.
* Double click the file to run it.
* You will see a bunch of ZIP archives starting to appear.
* Wait until the console window closes and don't close it yourself.

# What's new in 1.2?
* Fixed a bug where some antiviruses were marking the script as "harmful" and quarantined it.
* The script doesn't delete the folders anymore after creating ZIP files.
